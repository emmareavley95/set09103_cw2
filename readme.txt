Christmas Wishlist

- Open Levinux with Pipulate
- Select choice 2 to login
        login: tc
        password: foo
- Open your terminal
- Connect with your Levinux environment by taping this command below:
        $ ssh tc@localhost -p 2222
        (same password as above)
- Open my app folder in your terminal
        $ cd set09103_cw2/src
- Activate my webapp on Python
        $ python app.py
- Open your web browser at http://0.0.0.0:5000 and enjoy your visit!

PS: in case you could not log in with your username and password on
http://0.0.0.0:5000/login, go create another profile again on the root page.
